# Second Brain Organization Rules

## Directory Structure
- `research/` — web research, competitive analysis, trends
- `projects/` — active development projects
- `skills/` — custom OpenClaw skills
- `memory/` — daily logs (YYYY-MM-DD.md)
- `archive/` — completed/deprecated content

## File Naming
- Use kebab-case: `ai-agents-2026-trends.md`
- Include dates for time-sensitive content: `scholarship-deadlines-2026-03.md`
- Prefix with category for sorting: `project-kanban.md`, `research-tpusa.md`

## Cleanup Triggers
- Archive projects > 30 days inactive
- Consolidate memory files quarterly
- Remove duplicate research
- Update MEMORY.md weekly with distilled insights

## Automation
- Nightly cron: build + document + commit
- Heartbeat: periodic memory review + git status
- Session end: commit uncommitted changes
