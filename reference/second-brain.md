# Second Brain - Folder Structure

## Layout
```
workspace/
├── AGENTS.md           # How I operate
├── SOUL.md             # Who I am
├── USER.md             # About Brian
├── IDENTITY.md         # My identity
├── TOOLS.md            # Tool-specific notes
├── MEMORY.md           # Long-term curated memory
├── HEARTBEAT.md        # Periodic check config
├── memory/             # Daily logs
│   └── YYYY-MM-DD.md
├── kanban/             # Task tracking board
│   └── index.html
├── research/           # Research outputs
│   ├── scholarships-asu.md
│   ├── ai-community/
│   └── tpusa/
├── content/            # Content creation
│   ├── x-drafts/      # Draft posts for @swordtruth
│   ├── video-prompts/  # Grok Imagine prompts
│   └── tiktok-finds/   # Flagged TikTok content
├── projects/           # Coding projects
├── reference/          # Reference docs
│   └── second-brain.md
└── drafts/             # General drafts
```

## Principles
- Everything has a home
- Daily logs go in memory/
- Research goes in research/
- Content drafts go in content/
- Code projects go in projects/
- If unsure, ask Brian
